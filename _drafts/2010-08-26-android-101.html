So finally I got to work on the awesome Android development platform. One thing I really like about Android is that I can use my Eclipse set up for Android development. To be specific I am using Eclipse galileo, I got "Hello World"Android up and running in 30 mins (it takes about 15 mins for Android to configure itself).

Some of the issues I came across were
1) Could not find HelloAndroid.apk! :
I backtraced the issue and found out that the Android SDK was not able to find the java sdk, so I added the Java sdk path to the Windows class path and the error disappeared.

2) [2010-06-15 11:19:15 - Emulator] emulator: ERROR: unknown virtual device name: 'my_avd'
[2010-06-15 11:19:15 - Emulator] emulator: could not find virtual device named 'my_avd'
ans) I am using windows 7 as development platform (I know windows suck but  can't help), I had to copy the .android folder from
C:UsersAdministrator to C:Users<user> and the error was taken care of.

[Another way of doing it is change the permissions on the C:UsersAdministrator.android folder, give it full read write control]

So far so good, I am happy with my Android setup.

 

Debugging Tips:

Following are the steps to debug  javascript on Android simulator:

1) Using the command prompt go to the location where Android is installed, go to the tools directory.

2) Run the following command line command

>adb devices

3) This command will display all the android simulators running e.g.

>List of devices attache
>emulator-5554   device

4) Then use the following command to display the log

>adb -s emulator-5554 logcat Browser

5) logcat command displays the log, you have an option to filter the logs, I wanted only the browser output hence I used [Browser] as filter variable.
